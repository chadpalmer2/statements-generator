Using the Statements Generator

The following describes what information each uploaded file should have, and where they can be downloaded from. Note that all of these files, with the exception of 'Customer ID Filtering CSV', are to be downloaded directly from the relevant website and not modified. Modifying column names will almost certainly break the Statements Generator.

Production CSV

This file contains data on customer electricity generation in kWh.

NEPool: Download this file from "Individual Systems Production" at https://www.masscec-pts.com/#/reports/dashboard

PJM: Download this file from Assets > Generation > My Solar Generation at https://gats.pjm-eis.com/GATS2/AccountReports/AccountSolarGeneration/Filter.

Additional Production CSV (optional)

PJM data is compiled into monthly CSV files. To create statements for periods spanning up to three months, upload a CSV file for each.

Generator Details CSV

PJM CSV files do not include location data. To accurately price customer SRECs, you must include a separate CSV with this information. Download this file from Assets > Assets > My Generator Details at https://gats.pjm-eis.com/gats2/AccountReports/GeneratorDetails.

Customer ID Filtering CSV (optional)

If there is a list of customers in need of statements, a CSV file can be uploaded here to filter 'Production CSV' data. This is a file prepared by the end user, not a website. This file should have a single column with name "SystemID" consisting of system IDs for customers to generate statements for.
