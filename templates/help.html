Using the Statements Generator

The "Production CSV" file should be downloaded directly from the relevant website. Modifying column names will likely break the statement generator.

If NEPool data is being processed, the CSV should be downloaded from the "Individual Systems Production" panel at https://www.masscec-pts.com/#/reports/dashboard.

If PJM data is being processed, the CSV should be downloaded from the Assets > Generation > My Solar Generation page at https://gats.pjm-eis.com/GATS2/AccountReports/AccountSolarGeneration/Filter. You will additionally need to provide a CSV with location data on customers, which should be downloaded from the Assets > Assets > My Generator Details page at https://gats.pjm-eis.com/gats2/AccountReports/GeneratorDetails.

The "Customer ID CSV" file should have a column with name "SystemID" consisting of system IDs for customers to generate statements for. If left blank, no filtering will be done. Other columns in the CSV file will be ignored.
